О проекте M1kunRead

В разработке · BETA 1.0B

M1kunRead — удобная утилита для записи ISO на USB-накопители и создания загрузочных носителей. Просто выбрать образ, накопитель и дождаться проверки целостности.

Основные функции

  • Запись ISO файлов на USB
  • UEFI и Legacy режимы
  • Отображение прогресса
  • Поддержка файловых систем FAT32, NTFS, exFAT
  • ИСО на любые системы

Поддерживаемые системы

Windows

Windows 7, 8,
10, 11

Linux

Ubuntu, Fedora

, Arch Linux

Системные требования

  • Операционная система: Windows 7 или новее
  • USB-порт для подключения накопителя
  • Минимум 4 ГБ свободного места на USB-накопителе
  • Права администратора для записи
  • Поддержка USB 2.0 или новее
